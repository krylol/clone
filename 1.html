Comandos de git git init inicia repositorio git clone copia el repositorio creado en github git add -fileName guarda cambios en el archivo sin subirlos a git git add . guarda todos los cambios pendientes git commit -m "Mensaje que quiero subir" guarda los cambios en el main, queda subirlos git push sube el repositorio a github git pull descarga actualizaciones y cambios del proyecto git branch -M main cambia el nombre de master a main de la rama principal git branch ves las ramas que tienes en git, con git branch nombre_del_branch creas una nueva git checkout nombre_rama para cambiar de rama archivo .gitignore siempre se pone la carpeta nodemodules y lo que no queremos que se suba al repositorio Texto a√±adido para my_first_branch
